အဆင့်ဆင့် လမ်းညွှန်ချက်

ဝန်ထမ်း မိတ်ဆက်အစီအစဉ်

လုပ်ဖော်ကိုင်ဖက်များကို Rakuten Mobile သို့ မိတ်ဆက်ပြီး နှစ်ဦးစလုံး အပိုဆု Rakuten ပွိုင့်များ ရယူပါ။ ဤလမ်းညွှန်သည် ဖန်သားပြင်ဓာတ်ပုံများဖြင့် မိတ်ဆက်လုပ်ငန်းစဉ်ကို လမ်းပြပေးပါသည်။

အစီအစဉ် အကျဉ်းချုပ်

မိတ်ဆက်သူ

အောင်မြင်သော မိတ်ဆက်မှုတစ်ခုစီအတွက် အပိုဆုပွိုင့်များ ရရှိပါသည်

မိတ်ဆက်ခံရသူ

သင့်လုပ်ဖော်ကိုင်ဖက်လည်း စာရင်းသွင်းသောအခါ အပိုဆုပွိုင့်များ ရရှိပါသည်

အရည်အချင်းသတ်မှတ်ချက်များ:
  • မိတ်ဆက်သူနှင့် မိတ်ဆက်ခံရသူ နှစ်ဦးစလုံး အရည်အချင်းပြည့်မီသော ကုမ္ပဏီ၏ ဝန်ထမ်းများ ဖြစ်ရမည်
  • မိတ်ဆက်ခံရသူသည် Rakuten Mobile အသုံးပြုသူ အသစ်ဖြစ်ရမည်
  • လျှောက်ထားချိန်တွင် မိတ်ဆက်လင့်/ကုဒ်ကို အသုံးပြုရမည်
1

မိတ်ဆက်အစီအစဉ် စာမျက်နှာသို့ ဝင်ရောက်ပါ

သင့် Rakuten အကောင့်သို့ ဝင်ရောက်ပြီး ဝန်ထမ်း မိတ်ဆက်အစီအစဉ် စာမျက်နှာသို့ သွားပါ။ ၎င်းကို သင့်ကုမ္ပဏီ၏ အတွင်းပိုင်း portal မှ သို့မဟုတ် အလုပ်ရှင်မှ မျှဝေထားသော တိုက်ရိုက်လင့်မှ ရှာတွေ့နိုင်ပါသည်။

ဖန်သားပြင်ဓာတ်ပုံ: မိတ်ဆက်အစီအစဉ် ပင်မစာမျက်နှာ

သင့်မိတ်ဆက်လင့် သို့မဟုတ် ကုဒ်ကို ရယူပါ

မိတ်ဆက်စာမျက်နှာတွင် သင့်ကိုယ်ပိုင် မိတ်ဆက်လင့် သို့မဟုတ် ကုဒ်ကို တွေ့ရပါမည်။ ဤလင့်ကို ကူးယူပါ - သင့်လုပ်ဖော်ကိုင်ဖက်နှင့် မျှဝေရပါမည်။ မိတ်ဆက်လင့်တစ်ခုစီသည် သင့် Rakuten အကောင့်နှင့် ချိတ်ဆက်ထားပါသည်။

ဖန်သားပြင်ဓာတ်ပုံ: မိတ်ဆက်လင့်/ကုဒ် ထုတ်ပေးသည့် စာမျက်နှာ
Add image to: assets/guides/referral-step2.png
အကြံပြုချက်: မိတ်ဆက်လင့်ကို အဆင်ပြေသော နေရာတွင် သိမ်းဆည်းပါ (Notes app၊ bookmark စသည်) သင့်လုပ်ဖော်ကိုင်ဖက်က Rakuten Mobile အကြောင်း မေးသောအခါ အချိန်မရွေး မျှဝေနိုင်ပါသည်။

လုပ်ဖော်ကိုင်ဖက်နှင့် လင့်ကို မျှဝေပါ

မိတ်ဆက်လင့်ကို email၊ LINE၊ Slack သို့မဟုတ် မည်သည့် messaging app မဆို မှတစ်ဆင့် လုပ်ဖော်ကိုင်ဖက်ထံ ပေးပို့ပါ။ Rakuten Mobile စာရင်းသွင်းသောအခါ ဤလင့်ကို အသုံးပြုရမည်ကို နားလည်ကြောင်း သေချာပါစေ။

လုပ်ဖော်ကိုင်ဖက်က သင့်လင့်မှတစ်ဆင့် စာရင်းသွင်းပါ

သင့်လုပ်ဖော်ကိုင်ဖက်က မိတ်ဆက်လင့်ကို နှိပ်ပြီး Rakuten Mobile လျှောက်လွှာစာမျက်နှာသို့ ရောက်ရှိပါမည်။ ပုံမှန် စာရင်းသွင်းခြင်း လုပ်ငန်းစဉ်ကို ပြီးဆုံးအောင် လုပ်ဆောင်ပါ။ သင့်လင့်ကို အသုံးပြုသောအခါ မိတ်ဆက်ကုဒ်သည် အလိုအလျောက် အသုံးပြုပါမည်။

ဖန်သားပြင်ဓာတ်ပုံ: မိတ်ဆက်လင့်မှတစ်ဆင့် Rakuten Mobile စာရင်းသွင်းစာမျက်နှာ

အရေးကြီး: သင့်လုပ်ဖော်ကိုင်ဖက်က မိတ်ဆက်လင့်မှ ထွက်ခွာပြီး Rakuten Mobile သို့ တိုက်ရိုက်သွားပါက မိတ်ဆက်မှုကို ခြေရာခံ၍ မရနိုင်ပါ။ လျှောက်လွှာ စတင်ရန် သင့်လင့်ကို အသုံးပြုကြောင်း သေချာပါစေ။

လုပ်ဖော်ကိုင်ဖက်က Rakuten အကောင့် ဖန်တီးပါ (လိုအပ်ပါက)

သင့်လုပ်ဖော်ကိုင်ဖက်တွင် Rakuten အကောင့် မရှိသေးပါက အရင်ဖန်တီးရပါမည်။ အဆင့်ဆင့် ညွှန်ကြားချက်များအတွက် ကျွန်ုပ်တို့၏ Rakuten အကောင့် မှတ်ပုံတင်ခြင်း လမ်းညွှန် ကို ညွှန်ပြပေးပါ။

လျှောက်လွှာ ပြီးဆုံးပြီး အသက်သွင်းပါ

လုပ်ဖော်ကိုင်ဖက်က Rakuten Mobile လျှောက်လွှာကို ပြီးဆုံးအောင် လုပ်ဆောင်ပြီး SIM ကတ်ကို လက်ခံရရှိကာ ဝန်ဆောင်မှုကို အသက်သွင်းပါ။ မိတ်ဆက်ဆုကြေးသည် လိုင်းအသစ် အသက်သွင်းပြီး အရည်အချင်းပြည့်မီသော ကာလအတွင် အသက်ဝင်ပြီးနောက် အများအားဖြင့် ဆောင်ရွက်ပေးပါသည်။

  • ပွိုင့်များကို အသက်သွင်းပြီး ၁-၂ လအတွင်း ပုံမှန်အားဖြင့် ထည့်သွင်းပေးပါသည်

  • ပွိုင့်များကို Rakuten ဝန်ဆောင်မှုအားလုံးတွင် အသုံးပြုနိုင်ပါသည်

မေးလေ့ရှိသော မေးခွန်းများ

ကျွန်ုပ် ဘယ်နှစ်ယောက် မိတ်ဆက်နိုင်ပါသလဲ?
ပုံမှန်အားဖြင့် မိတ်ဆက်မှု အရေအတွက် ကန့်သတ်ချက် မရှိပါ၊ သို့သော် ကန့်သတ်ချက် ရှိမရှိ သင့်ကုမ္ပဏီ၏ အစီအစဉ် စည်းကမ်းချက်များကို စစ်ဆေးပါ။
Rakuten Mobile ရှိပြီးသား သူကို မိတ်ဆက်နိုင်ပါသလား?
မရပါ၊ မိတ်ဆက်ခံရသူသည် Rakuten Mobile အသုံးပြုသူ အသစ်ဖြစ်ရပါမည်။ လက်ရှိ အသုံးပြုသူများသည် အရည်အချင်း မပြည့်မီပါ။
လုပ်ဖော်ကိုင်ဖက်က စာရင်းသွင်းပြီးနောက် ပယ်ဖျက်ပါက ဘယ်လိုလဲ?
မိတ်ဆက်ခံရသူက အရည်အချင်းပြည့်မီသော ကာလမတိုင်မီ ပယ်ဖျက်ပါက အပိုဆုပွိုင့်များကို ပြန်လည်ရုပ်သိမ်းခြင်း သို့မဟုတ် မပေးခြင်း ဖြစ်နိုင်ပါသည်။ သတ်မှတ်ထားသော ကာလအတွင်း ဝန်ဆောင်မှု အသက်ဝင်နေရပါမည်။
အပိုဆုပွိုင့်များ သက်တမ်းကုန်ပါသလား?
မိတ်ဆက်ဆု ပွိုင့်များသည် သက်တမ်းကုန်ရက် ရှိသော "ကာလကန့်သတ် ပွိုင့်များ" (期間限定ポイント) ဖြစ်နိုင်ပါသည်။ သင့် Rakuten Point Club တွင် ပွိုင့်အသေးစိတ်ကို စစ်ဆေးပါ။
