ပင်မစာမျက်နှာ SIM နှိုင်းယှဉ်ခြင်း ခရက်ဒစ်ကတ် လမ်းညွှန်

အဆင့်ဆင့် လမ်းညွှန်

Rakuten အကောင့် မှတ်ပုံတင်နည်း

Rakuten အကောင့် (Rakuten ID) သည် Rakuten Mobile၊ Rakuten Card၊ Rakuten Ichiba နှင့် Rakuten ဝန်ဆောင်မှုအားလုံးအတွက် လိုအပ်ပါသည်။ မှတ်ပုံတင်ခြင်း အခမဲ့ဖြစ်ပြီး ၅ မိနစ်ခန့် ကြာပါသည်။

မစတင်မီ

လိုအပ်သောအရာများ:
  • အီးမေးလ်လိပ်စာ
  • ဖုန်းနံပါတ် (အတည်ပြုရန်)
  • ၅ မိနစ်ခန့်
သိထားသင့်သည်:
  • စာမျက်နှာအောက်ခြေရှိ Language ခလုတ်ဖြင့် ဘာသာစကားပြောင်းနိုင်သည်
  • အောက်ပါ ဖန်သားပြင်ဓာတ်ပုံများသည် အင်္ဂလိပ်ဘာသာ ဖြစ်သည်
  • မည်သည့်ဘာသာစကားဖြင့်မဆို လုပ်ငန်းစဉ်တူညီသည်
1

Rakuten အကောင့်ဝင်စာမျက်နှာသို့ သွားပြီး "Create a new Rakuten account" ကို နှိပ်ပါ

Rakuten ဝန်ဆောင်မှု အကောင့်ဝင်စာမျက်နှာသို့ သွားပါ။ စာမျက်နှာအောက်ခြေတွင် ပန်းရောင် "Create a new Rakuten account (free)" ခလုတ်ကို နှိပ်ပါ။

2

အီးမေးလ်လိပ်စာ ထည့်သွင်းပါ

"Email (Required)" ကွက်လပ်တွင် သင့်အီးမေးလ်ကို ထည့်ပါ။ ဤအီးမေးလ်ကို အကောင့်ဝင်ရန် အသုံးပြုပါမည်။ "Send verification code" ကို နှိပ်ပါ။

3

အတည်ပြုကုဒ် ထည့်သွင်းပါ

Rakuten မှ ဂဏန်း ၆ လုံးပါ အတည်ပြုကုဒ်ကို သင့်အီးမေးလ်သို့ ပေးပို့ပါမည်။ ကုဒ်ကို "Verification code (Required)" ကွက်လပ်တွင် ထည့်ပါ။ ကုဒ် ၁၀ မိနစ်အတွင်း အကျုံးဝင်ပါသည်။

ကုဒ်မရရှိပါက: spam/junk ဖိုဒါကို စစ်ဆေးပါ။ အချိန်ပြည့်ပြီးနောက် "Resend verification code" ကို နှိပ်နိုင်ပါသည်။
4

Username, password နှင့် အမည်ထည့်ပါ

အတည်ပြုပြီးနောက် အကောင့်အချက်အလက်များ ဖြည့်ပါ:

  • Username — မူလအားဖြင့် သင့်အီးမေးလ်
  • Password — ခိုင်မာသော password ထည့်ပါ
  • Re-enter password — ထပ်မံထည့်၍ အတည်ပြုပါ
  • First name / Last name — မှတ်ပုံတင်ပေါ်ရှိ အမည်အတိုင်း ထည့်ပါ

"Continue to confirmation page" ကို နှိပ်ပါ။

နိုင်ငံခြားသားများအတွက်: သင့်နေထိုင်ခွင့်ကတ် (在留カード) ပေါ်ရှိ အမည်အတိုင်း ထည့်ပါ။
5

အချက်အလက်များ အတည်ပြုပြီး မှတ်ပုံတင်ခြင်း ပြီးမြောက်ပါ

အတည်ပြုစာမျက်နှာတွင် အချက်အလက်များကို စစ်ဆေးပါ။ မှန်ကန်ပါက ပန်းရောင် "Complete registration" ခလုတ်ကို နှိပ်ပါ။

6

မှတ်ပုံတင်ခြင်း ပြီးမြောက်ပါပြီ!

သင့် Rakuten အကောင့် အသုံးပြုနိုင်ပါပြီ:

  • Rakuten Mobile — ဖုန်းအစီအစဉ် မှာယူရန်
  • Rakuten Card — ခရက်ဒစ်ကတ် လျှောက်ရန်
  • Rakuten Ichiba — အွန်လိုင်းစျေးဝယ်ရန်
  • Rakuten Pay — ငွေသားမဲ့ ငွေပေးချေရန်
  • Rakuten Points — ဝန်ဆောင်မှုအားလုံးတွင် အမှတ်စုဆောင်း/အသုံးပြုရန်
