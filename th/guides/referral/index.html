คู่มือทีละขั้นตอน

โปรแกรมแนะนำพนักงาน

แนะนำเพื่อนร่วมงานให้ใช้ Rakuten Mobile แล้วทั้งคู่จะได้รับคะแนน Rakuten โบนัส คู่มือนี้จะพาคุณผ่านขั้นตอนการแนะนำพร้อมภาพหน้าจอ

ภาพรวมโปรแกรม

ผู้แนะนำ

คุณจะได้รับคะแนนโบนัสสำหรับการแนะนำที่สำเร็จแต่ละครั้ง

ผู้ถูกแนะนำ

เพื่อนร่วมงานของคุณก็จะได้รับคะแนนโบนัสเมื่อสมัคร

คุณสมบัติ:
  • ทั้งผู้แนะนำและผู้ถูกแนะนำต้องเป็นพนักงานของบริษัทที่มีคุณสมบัติ
  • ผู้ถูกแนะนำต้องเป็นผู้ใช้ Rakuten Mobile รายใหม่
  • ต้องใช้ลิงก์/รหัสแนะนำในขณะสมัคร
1

เข้าสู่หน้าโปรแกรมแนะนำ

เข้าสู่ระบบบัญชี Rakuten ของคุณแล้วไปที่หน้าโปรแกรมแนะนำพนักงาน โดยปกติจะพบได้ผ่านพอร์ทัลภายในของบริษัทหรือลิงก์โดยตรงที่นายจ้างแชร์ให้

ภาพหน้าจอ: หน้าแรกโปรแกรมแนะนำ
Add image to: assets/guides/referral-step1.png
2

รับลิงก์หรือรหัสแนะนำของคุณ

ในหน้าแนะนำ คุณจะพบลิงก์หรือรหัสแนะนำเฉพาะของคุณ คัดลอกลิงก์นี้ - คุณจะแชร์กับเพื่อนร่วมงาน ลิงก์แนะนำแต่ละอันผูกกับบัญชี Rakuten ของคุณ

ภาพหน้าจอ: หน้าสร้างลิงก์/รหัสแนะนำ
Add image to: assets/guides/referral-step2.png
เคล็ดลับ: บันทึกลิงก์แนะนำไว้ในที่สะดวก (แอป Notes, บุ๊กมาร์ก ฯลฯ) เพื่อให้คุณแชร์ได้ทุกเมื่อที่เพื่อนร่วมงานถามเกี่ยวกับ Rakuten Mobile
3

แชร์ลิงก์กับเพื่อนร่วมงาน

ส่งลิงก์แนะนำให้เพื่อนร่วมงานผ่านอีเมล LINE Slack หรือแอปส่งข้อความใดก็ได้ ให้แน่ใจว่าพวกเขาเข้าใจว่าต้องใช้ลิงก์นี้เมื่อสมัคร Rakuten Mobile

ภาพหน้าจอ: การแชร์ลิงก์แนะนำ
Add image to: assets/guides/referral-step3.png
4

เพื่อนร่วมงานสมัครผ่านลิงก์ของคุณ

เพื่อนร่วมงานคลิกลิงก์แนะนำของคุณ ซึ่งจะพาไปยังหน้าสมัคร Rakuten Mobile พวกเขาดำเนินการสมัครตามปกติ รหัสแนะนำจะถูกใช้โดยอัตโนมัติเมื่อใช้ลิงก์ของคุณ

ภาพหน้าจอ: หน้าสมัคร Rakuten Mobile ผ่านลิงก์แนะนำ
Add image to: assets/guides/referral-step4.png
สำคัญ: หากเพื่อนร่วมงานออกจากลิงก์แนะนำแล้วไป Rakuten Mobile โดยตรง การแนะนำอาจไม่ถูกติดตาม ให้แน่ใจว่าพวกเขาใช้ลิงก์ของคุณเพื่อเริ่มสมัคร
5

เพื่อนร่วมงานสร้างบัญชี Rakuten (ถ้าจำเป็น)

หากเพื่อนร่วมงานยังไม่มีบัญชี Rakuten จะต้องสร้างก่อน แนะนำให้ดูคู่มือสมัครบัญชี Rakuten สำหรับคำแนะนำทีละขั้นตอน

ภาพหน้าจอ: การสร้างบัญชี Rakuten ในขั้นตอนการแนะนำ
Add image to: assets/guides/referral-step5.png
6

ดำเนินการสมัครและเปิดใช้งาน

เพื่อนร่วมงานดำเนินการสมัคร Rakuten Mobile เสร็จสิ้น รับซิมการ์ด และเปิดใช้งานบริการ โบนัสแนะนำมักจะถูกดำเนินการหลังจากสายใหม่เปิดใช้งานและใช้งานครบตามระยะเวลาที่กำหนด

ภาพหน้าจอ: การยืนยันสมัครเสร็จสิ้น
Add image to: assets/guides/referral-step6.png
7

รับคะแนนโบนัส

หลังจากสายของเพื่อนร่วมงานเปิดใช้งานและผ่านระยะเวลาที่กำหนดแล้ว ทั้งคู่จะได้รับคะแนน Rakuten โบนัส คุณสามารถตรวจสอบสถานะในบัญชี Rakuten ที่หน้าโปรแกรมแนะนำ

  • คะแนนมักจะถูกเพิ่มภายใน 1-2 เดือนหลังเปิดใช้งาน
  • ตรวจสอบ Rakuten Point Club สำหรับการเพิ่มคะแนน
  • คะแนนสามารถใช้ได้ในบริการ Rakuten ทั้งหมด
ภาพหน้าจอ: การยืนยันคะแนนที่ได้รับ
Add image to: assets/guides/referral-step7.png

คำถามที่พบบ่อย

ฉันสามารถแนะนำได้กี่คน?
โดยปกติไม่มีจำกัดจำนวนการแนะนำ แต่ตรวจสอบเงื่อนไขโปรแกรมเฉพาะของบริษัทคุณว่ามีขีดจำกัดหรือไม่
ฉันสามารถแนะนำคนที่มี Rakuten Mobile อยู่แล้วได้ไหม?
ไม่ได้ ผู้ถูกแนะนำต้องเป็นผู้ใช้ Rakuten Mobile รายใหม่ ผู้ใช้ปัจจุบันไม่มีคุณสมบัติ
ถ้าเพื่อนร่วมงานยกเลิกหลังสมัครล่ะ?
หากผู้ถูกแนะนำยกเลิกก่อนระยะเวลาที่กำหนด คะแนนโบนัสอาจถูกเพิกถอนหรือไม่ได้รับ บริการต้องยังคงใช้งานอยู่ตลอดระยะเวลาที่กำหนด
คะแนนโบนัสหมดอายุไหม?
คะแนนโบนัสแนะนำอาจเป็น "คะแนนจำกัดเวลา" (期間限定ポイント) ที่มีวันหมดอายุ ตรวจสอบรายละเอียดคะแนนใน Rakuten Point Club ของคุณ

ยังไม่มี Rakuten Mobile?

สมัครด้วยตัวเองก่อนแล้วเริ่มแนะนำเพื่อนร่วมงานเพื่อรับคะแนน
