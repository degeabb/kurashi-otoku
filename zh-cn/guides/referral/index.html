员工推荐计划

推荐同事使用乐天Mobile，双方都能获得奖励乐天积分。本指南将通过截图引导您完成推荐流程。

计划概览

推荐人

每次成功推荐可获得奖励积分

被推荐人

您的同事在注册时也能获得奖励积分

资格条件：
  • 推荐人和被推荐人都必须是符合条件的公司员工
  • 被推荐人必须是乐天Mobile的新用户
  • 申请时必须使用推荐链接/代码
1

进入推荐计划页面

登录您的乐天账户并前往员工推荐计划页面。通常可以通过公司内部门户网站或雇主提供的直接链接找到。

截图：推荐计划首页
Add image to: assets/guides/referral-step1.png
2

获取您的推荐链接或代码

在推荐页面上，您会找到专属的推荐链接或代码。复制此链接——您将与同事分享。每个推荐链接都与您的乐天账户绑定。

截图：推荐链接/代码生成页面
Add image to: assets/guides/referral-step2.png
提示： 将推荐链接保存在方便的地方（备忘录、书签等），这样当同事询问乐天Mobile时可以随时分享。
3

与同事分享链接

通过电子邮件、LINE、Slack或任何通讯应用将推荐链接发送给同事。确保他们了解在注册乐天Mobile时需要使用这个特定链接。

截图：分享推荐链接
Add image to: assets/guides/referral-step3.png
4

同事通过您的链接注册

您的同事点击推荐链接，进入乐天Mobile申请页面。他们完成正常的注册流程。使用您的链接时，推荐代码会自动应用。

截图：通过推荐链接的乐天Mobile注册页面
Add image to: assets/guides/referral-step4.png
重要： 如果同事离开推荐链接直接前往乐天Mobile网站，推荐可能无法追踪。请确保他们使用您的链接开始申请。
5

同事创建乐天账户（如需要）

如果同事还没有乐天账户，需要先创建一个。请引导他们参考我们的乐天账户注册指南，获取逐步说明。

截图：推荐流程中的乐天账户创建
Add image to: assets/guides/referral-step5.png
6

完成申请并激活

同事完成乐天Mobile申请，收到SIM卡并激活服务。推荐奖励通常在新线路激活并活跃一段时间后处理。

截图：申请完成确认
Add image to: assets/guides/referral-step6.png
7

领取奖励积分

同事的线路激活且经过资格期后，双方都将获得奖励乐天积分。您可以在乐天账户的推荐计划页面查看状态。

  • 积分通常在激活后1-2个月内到账
  • 在乐天Point Club查看积分到账情况
  • 积分可在所有乐天服务中使用

常见问题

我可以推荐多少人？
通常没有推荐人数限制，但请查看您公司的具体计划条款是否有上限。
我可以推荐已经有乐天Mobile的人吗？
不可以，被推荐人必须是乐天Mobile的新用户。现有用户不符合资格。
如果同事注册后取消怎么办？
如果被推荐人在资格期前取消，奖励积分可能会被撤回或不予发放。服务必须在规定期间内保持活跃。
奖励积分会过期吗？
推荐奖励积分可能是"期间限定积分"（期間限定ポイント），有到期日。请在乐天Point Club查看积分详情。

还没有乐天Mobile吗？

先自己注册，然后开始推荐同事赚取积分。
