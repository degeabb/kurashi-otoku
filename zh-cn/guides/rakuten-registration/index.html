如何注册乐天账号

乐天账号（乐天ID）是使用乐天Mobile、乐天信用卡、乐天市场等所有乐天服务的必要条件。注册免费，约5分钟即可完成。

开始之前

您需要准备：
  • 电子邮箱地址
  • 手机号码（用于验证）
  • 约5分钟时间
注意事项：
  • 可使用页面底部的Language按钮切换语言
  • 以下截图为英文界面
  • 无论使用何种语言，流程相同
1

前往乐天登录页面，点击"Create a new Rakuten account"

前往任何乐天服务的登录页面。在页面底部，点击粉色的"Create a new Rakuten account (free)"按钮开始注册。

2

输入电子邮箱地址

在注册页面，于"Email (Required)"栏位输入您的邮箱。此邮箱将用于登录账号。乐天会发送验证码到此地址。点击"Send verification code"继续。

3

输入验证码

乐天会发送6位数验证码到您的邮箱。查看收件箱，找到验证码后输入至"Verification code (Required)"栏位。验证码有效期为10分钟。点击"Verify email address"继续。

没有收到验证码？请检查垃圾邮件文件夹。倒计时结束后可点击"Resend verification code"重新发送。
4

设置用户名、密码和姓名

验证完成后，填写账号信息：

  • Username — 默认为您的电子邮箱
  • Password — 设置强密码（建议"Medium"强度以上）
  • Re-enter password — 再次输入确认
  • First name / Last name — 输入与证件相同的姓名

完成后点击"Continue to confirmation page"。

外国人提示：请使用与在留卡上完全相同的姓名。英文界面可直接输入罗马字母。
5

确认信息并完成注册

在确认页面检查所有信息：电子邮箱、用户名、姓名。确认无误后点击粉色的"Complete registration"按钮。

6

注册完成！

您的乐天账号已可使用。可用于以下服务：

  • 乐天Mobile — 申请手机套餐
  • 乐天信用卡 — 申请信用卡
  • 乐天市场 — 在线购物
  • 乐天Pay — 移动支付
  • 乐天积分 — 跨服务积累与使用
