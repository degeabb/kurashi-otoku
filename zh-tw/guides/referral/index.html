員工推薦計劃

推薦同事使用樂天Mobile，雙方都能獲得獎勵樂天積分。本指南將透過截圖引導您完成推薦流程。

計劃概覽

推薦人

每次成功推薦可獲得獎勵積分

被推薦人

您的同事在註冊時也能獲得獎勵積分

資格條件：
  • 推薦人和被推薦人都必須是符合條件的公司員工
  • 被推薦人必須是樂天Mobile的新用戶
  • 申請時必須使用推薦連結/代碼
1

進入推薦計劃頁面

登入您的樂天帳戶並前往員工推薦計劃頁面。通常可以透過公司內部入口網站或雇主提供的直接連結找到。

截圖：推薦計劃首頁

2

取得您的推薦連結或代碼

在推薦頁面上，您會找到專屬的推薦連結或代碼。複製此連結——您將與同事分享。每個推薦連結都與您的樂天帳戶綁定。

截圖：推薦連結/代碼生成頁面
Add image to: assets/guides/referral-step2.png
提示： 將推薦連結保存在方便的地方（備忘錄、書籤等），這樣當同事詢問樂天Mobile時可以隨時分享。
3

與同事分享連結

透過電子郵件、LINE、Slack或任何通訊應用將推薦連結發送給同事。確保他們了解在註冊樂天Mobile時需要使用這個特定連結。

截圖：分享推薦連結
Add image to: assets/guides/referral-step3.png
4

同事透過您的連結註冊

您的同事點擊推薦連結，進入樂天Mobile申請頁面。他們完成正常的註冊流程。使用您的連結時，推薦代碼會自動套用。

截圖：透過推薦連結的樂天Mobile註冊頁面
Add image to: assets/guides/referral-step4.png
重要： 如果同事離開推薦連結直接前往樂天Mobile網站，推薦可能無法追蹤。請確保他們使用您的連結開始申請。
5

同事建立樂天帳戶（如需要）

如果同事還沒有樂天帳戶，需要先建立一個。請引導他們參考我們的樂天帳戶註冊指南，獲取逐步說明。

截圖：推薦流程中的樂天帳戶建立

6

完成申請並啟用

同事完成樂天Mobile申請，收到SIM卡並啟用服務。推薦獎勵通常在新線路啟用並活躍一段時間後處理。

截圖：申請完成確認
Add image to: assets/guides/referral-step6.png
7

領取獎勵積分

同事的線路啟用且經過資格期後，雙方都將獲得獎勵樂天積分。您可以在樂天帳戶的推薦計劃頁面查看狀態。

  • 積分通常在啟用後1-2個月內入帳
  • 在樂天Point Club查看積分入帳情況
  • 積分可在所有樂天服務中使用

常見問題

我可以推薦多少人？
通常沒有推薦人數限制，但請查看您公司的具體計劃條款是否有上限。
我可以推薦已經有樂天Mobile的人嗎？
不可以，被推薦人必須是樂天Mobile的新用戶。現有用戶不符合資格。
如果同事註冊後取消怎麼辦？
如果被推薦人在資格期前取消，獎勵積分可能會被撤回或不予發放。服務必須在規定期間內保持活躍。
獎勵積分會過期嗎？
推薦獎勵積分可能是「期間限定積分」（期間限定ポイント），有到期日。請在樂天Point Club查看積分詳情。

還沒有樂天Mobile嗎？

先自己註冊，然後開始推薦同事賺取積分。
