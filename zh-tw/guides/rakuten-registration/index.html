如何註冊樂天帳號

樂天帳號（樂天ID）是使用樂天Mobile、樂天信用卡、樂天市場等所有樂天服務的必要條件。註冊免費，約5分鐘即可完成。

開始之前

您需要準備：
  • 電子郵件地址
  • 手機號碼（用於驗證）
  • 約5分鐘時間
注意事項：
  • 可使用頁面底部的Language按鈕切換語言
  • 以下截圖為英文介面
  • 無論使用何種語言，流程相同
1

前往樂天登入頁面，點擊「Create a new Rakuten account」

前往任何樂天服務的登入頁面。在頁面底部，點擊粉色的「Create a new Rakuten account (free)」按鈕開始註冊。

2

輸入電子郵件地址

在註冊頁面，於「Email (Required)」欄位輸入您的電子郵件。此郵件將用於登入帳號。樂天會發送驗證碼到此地址。點擊「Send verification code」繼續。

3

輸入驗證碼

樂天會發送6位數驗證碼到您的郵件。查看收件匣，找到驗證碼後輸入至「Verification code (Required)」欄位。驗證碼有效期為10分鐘。點擊「Verify email address」繼續。

沒有收到驗證碼？ 請檢查垃圾郵件資料夾。倒數計時結束後可點擊「Resend verification code」重新發送。
4

設定使用者名稱、密碼和姓名

驗證完成後，填寫帳號資訊：

  • Username — 預設為您的電子郵件
  • Password — 設定強密碼（建議「Medium」強度以上）
  • Re-enter password — 再次輸入確認
  • First name / Last name — 輸入與證件相同的姓名

完成後點擊「Continue to confirmation page」。

外國人提示：請使用與在留卡上完全相同的姓名。英文介面可直接輸入羅馬字母。
5

確認資訊並完成註冊

在確認頁面檢查所有資訊：電子郵件、使用者名稱、姓名。確認無誤後點擊粉色的「Complete registration」按鈕。

6

註冊完成！

您的樂天帳號已可使用。可用於以下服務：

  • 樂天Mobile — 申請手機方案
  • 樂天信用卡 — 申請信用卡
  • 樂天市場 — 線上購物
  • 樂天Pay — 行動支付
  • 樂天點數 — 跨服務累積與使用
